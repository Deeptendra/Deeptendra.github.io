# Project Documents

This directory contains documents related to various projects.

## How to Upload/Update Documents

To upload or update a document:

1. Add your PDF or document file to this directory with a descriptive, permanent filename
2. The document will be accessible at: `https://deeptendra.github.io/documents/[filename]`
3. Update the link in `index.html` if needed
